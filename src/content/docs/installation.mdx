BeeCharts is designed to be a plug-and-play solution for beautiful, interactive charts in your Next.js projects. You don't need any complex setup or high-end configuration—just follow these simple steps to get started with stunning data visualizations in minutes.

## Steps
<Steps>
  <Step>
    <StepTitle>Install Recharts</StepTitle>
    <StepContent>
      <StepDescription>
        Install Recharts, the charting library that powers BeeCharts. This is a required dependency for all chart components.
        
        Learn more at the <Link href="https://recharts.github.io/en-US/guide/installation" _blank>Recharts installation guide</Link>.
      </StepDescription>
      <CommandBlock commands={["recharts"]} />
    </StepContent>
  </Step>
  
  <Step>
    <StepTitle>Setup shadcn/ui</StepTitle>
    <StepContent>
      <StepDescription>
        Initialize shadcn/ui in your project. This sets up Tailwind CSS, theme configuration, and component structure. Skip this step if you already have shadcn/ui installed.
        
        Check out the <Link href="https://ui.shadcn.com/docs/installation/next" _blank>shadcn/ui Next.js installation docs</Link> for more details.
      </StepDescription>
      <CliBlock commands={["init"]} />
    </StepContent>
  </Step>
  
  <Step>
    <StepTitle>Add BeeCharts Components</StepTitle>
    <StepContent>
      <StepDescription>
        Add chart components to your project using the CLI. Replace `{chart-name}` with your desired chart type (e.g., `area-chart`, `bar-chart`, `line-chart`, `pie-chart`). The CLI will install all necessary files and dependencies.
      </StepDescription>
      <CliBlock commands={["@evilcharts/{chart-name}"]} />
    </StepContent>
  </Step>

  <Step>
    <StepTitle>Optional — Chart recipes (BI data helpers)</StepTitle>
    <StepContent>
      <StepDescription>
        For histogram, Pareto, heatmap, bullet, gauge, and box plot patterns, install the shared data helpers. See [Chart recipes](/docs/chart-recipes) for usage and example block names.
      </StepDescription>
      <CliBlock commands={["@evilcharts/chart-recipes"]} />
    </StepContent>
  </Step>
</Steps>
